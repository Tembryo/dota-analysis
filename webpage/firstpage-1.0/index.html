About The Prototype

This is an early-stage protoype for an application that will take in replays and automatically generate an overview of the gameplay. The idea is to allow you to review a game quickly and focus on the important aspects without having to watch the whole replay.

To demonstrate our approach we took the first 10 minutes of a (Monkey)Business vs NiP game (Frankfurt majors qualifier WB Finals game 1) and made the visualisation above. By clicking on different heros you can see the path each one took through the game and the fights they were engaged in.

The project is still obviously a work in progress - with lots more features planned - so stay tuned for updates and let us know if you have any feedback. We will be running a closed beta from the 23rd of November 2015 where we will allow you to submit a replay for us to analyse. You can sign up to the beta here:
